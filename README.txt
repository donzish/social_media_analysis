Questo progetto è stato sviluppato con lo scopo di effettuare un'analisi del canale youtube Luke Barousse (link:https://www.youtube.com/c/LukeBarousse).
Questo canale tratta temi riguardanti la data science ed in generale il mondo dei dati.
Dopo una iniziale fase di download del dataset attraverso l'uso delle API offerte da Youtube (notebook data_download), sono state eseguite una social network analysis ed una sentiment analysis (notebook relativi).
All'interno della cartella è presente il dataset scaricato in formato .csv, il report ed una presentazione in formato .pdf.
L'intero progetto è stato sviluppato in Python.
